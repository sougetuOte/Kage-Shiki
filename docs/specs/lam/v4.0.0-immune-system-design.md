# 免疫系アーキテクチャ設計（LAM v4.0.0）

## 概要

免疫系アーキテクチャの設計。3 つのフィードバックループで構成される。

## アーキテクチャ

### Loop 1: リアルタイム防御（PreToolUse hook）
- ファイルパスベースの PG/SE/PM 分類
- PM級変更のブロック（AUDITING フェーズ時）
- 即座のフィードバック

### Loop 2: バッチ検証（`/full-review` 自動ループ）
- 4 エージェントによる並列監査
- Green State 5 条件の検証
- 最大 5 イテレーションの自動ループ

### Loop 3: 学習（TDD 内省パイプライン）
- テスト失敗→成功パターンの自動記録
- パターン閾値（3回）到達時のルール候補生成
- PM級承認によるルール昇格
